О проекте

Задача

Создать сайт-визитку для IT-агентства Synapse Hub, который не просто рассказывает об услугах, а демонстрирует техническое превосходство через интерактивные элементы. Каждый скролл — намеренный, каждая анимация — взвешенная и профессиональная.

Дизайн-система «Silicon Void» — тёмная премиальная эстетика с глубоким фоном slate-950, фиолетовым акцентом #6366f1, контрастной типографикой Inter + Instrument Serif + JetBrains Mono и noise-текстурой через SVG feTurbulence.

Ключевые секции
⚡

Hero с двойной типографикой

Крупный заголовок sans + массивный serif italic. GSAP-анимация staggered fade-up. Интерактивное поле заявки прямо в Hero. Анимированные карточки: код, бот-чат, метрики.

🔄

Интерактивный Pipeline

Пошаговый визуализатор процесса (Заявка → Аудит → Разработка → Запуск). Переключение контента в реальном времени. Анимированный бот-чат, граф данных, код-свитчер.

📊

ROI-калькулятор

Клиент вводит параметры бизнеса — калькулятор показывает экономию от автоматизации. Визуализация результатов, конверсия в заявку одним кликом.

🤖

AI-чат ассистент

Встроенный виджет чата с AI. Отвечает на вопросы посетителей о сервисах, ценах и сроках. Плавающая кнопка с анимацией.

💼

Портфолио кейсов

Карточки реальных проектов с описанием, стеком технологий и результатами. Hover-эффекты, accent-подсветка активного кейса.

💰

Тарифная сетка

Три тарифа с выделенным популярным планом: accent gradient, glow-тень, увеличенный масштаб. CTA-кнопки с magnetic-эффектом.
